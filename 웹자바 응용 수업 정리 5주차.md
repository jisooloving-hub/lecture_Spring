## 1. 로깅 기능 설정

### Log4j

- **Log4j (Log for Java)**: 아파치 소프트웨어 라이선스로 배포되는 Java 로깅 프레임워크
- 안정적이고 신속하며 유연함
- 시스템 성능에 큰 영향 없이 선택적 로그 기록 가능
- 공식 문서: https://logging.apache.org/log4j/2.x/index.html

---

### Log4j 로깅 레벨

| 레벨 | 설명 |
| --- | --- |
| **FATAL** | 조기 종료를 유발하는 심각한 오류 |
| **ERROR** | 기타 런타임 오류 또는 예기치 않은 상태 |
| **WARN** | 사용 안 되는 API 사용, 예기치 않은 런타임 상황 경고 |
| **INFO** | 시작, 종료 같은 런타임 이벤트 메시지 |
| **DEBUG** | 디버그 용도의 시스템 흐름 상세 정보 |
| **TRACE** | 가장 하위 레벨, 모든 로그에 대한 상세 정보 |

> 🔺 **심각도 순서**: `TRACE < DEBUG < INFO < WARN < ERROR < FATAL`
> 
> 
> ※ 로깅 레벨은 프레임워크마다 다를 수 있음
> 

---

### 실습: Log4j 설정 (`src/main/resources/log4j2.xml`)

```xml
<?xml version="1.0" encoding="UTF-8"?>
<Configuration xmlns="https://logging.apache.org/xml/ns" ...>
  <Appenders>
    <Console name="console">
      <PatternLayout pattern="%d{yyyy-MM-dd HH:mm:ss} %-5p %c{-10}:%L - %m%n"/>
    </Console>
  </Appenders>
  <Loggers>
    <Root level="INFO" additivity="false">
      <AppenderRef ref="console"/>
    </Root>
    <Logger name="com.springmvc" level="DEBUG"/>
    <Logger name="org.springframework.core" level="INFO"/>
    <Logger name="org.springframework.beans" level="INFO"/>
    <Logger name="org.springframework.context" level="INFO"/>
    <Logger name="org.springframework.web" level="DEBUG"/>
    <Logger name="org.springframework.security" level="DEBUG"/>
    <Logger name="org.springframework.jdbc" level="DEBUG"/>
  </Loggers>
</Configuration>
```

- PatternLayout 참고: https://logging.apache.org/log4j/2.x/manual/pattern-layout.html

---

### Slf4j를 이용한 로깅

- **Slf4j (Simple Logging Facade for Java)**: 다양한 로깅 프레임워크에 대한 추상화 API 제공
    - `java.util.logging`, `logback`, `log4j` 등을 지원
- 배포 시점에 원하는 로깅 프레임워크를 플러그인할 수 있음
- **스프링에서 기본 로깅 API로 사용**

```java
import org.slf4j.Logger;
import org.slf4j.LoggerFactory;

public class LoginController {
    private final Logger log = LoggerFactory.getLogger(LoginController.class);

    public String index() {
        this.log.debug("index");
        return "index";
    }
}
```

## 2. 데이터베이스 설치

### H2 데이터베이스

- Java로 개발된 **관계형 데이터베이스 관리 시스템 (RDBMS)**
- 공식 사이트: https://www.h2database.com/html/main.html

**H2의 주요 특징:**
- 매우 빠른 성능
- JDBC API 지원
- 오픈소스 소프트웨어
- 내장 DB, 서버 모드, **in-memory DB** 지원
- 브라우저 기반 Console 제공
- 약 2.5 MB의 작은 크기
- 표준 SQL의 일부만 지원 (제한 있음)
- 개발환경, 임베디드 애플리케이션, 휘발성 데이터 관리에 적합

> 유사 DB: Apache Derby (https://db.apache.org/derby/)
> 

---

## 3. JDBC 연동을 위한 환경 설정

### 환경설정 파일 구성 개요

| 파일 | 역할 |
| --- | --- |
| `WEB-INF/spring/servlet-context.xml` | JdbcTemplate 설정 |
| `WEB-INF/spring/root-context.xml` | db.properties 로드, DataSource 생성, DB 초기화 |
| `src/main/resources/db.properties` | 데이터베이스 연결 정보 설정 |
| `src/main/resources/db/create-db.sql` | 테이블 생성 스크립트 |
| `src/main/resources/db/insert-data.sql` | 초기 데이터 삽입 스크립트 |
| `WEB-INF/web.xml` | H2Console 서블릿 설정 |

---

### 데이터베이스 연결정보 설정 (`db.properties`)

```
datasource.driverClassName=org.h2.Driver
datasource.url=jdbc:h2:file:C:/webjavaapp/{학번}/data
datasource.username=admin
datasource.password=manager
```

---

### 테이블 생성 스크립트 (`db/create-db.sql`)

```sql
CREATE TABLE IF NOT EXISTS book(
    b_bookId        VARCHAR(10)  NOT NULL,
    b_name          VARCHAR(30),
    b_unitPrice     INTEGER,
    b_author        VARCHAR(50),
    b_description   TEXT,
    b_publisher     VARCHAR(20),
    b_category      VARCHAR(20),
    b_unitsInStock  LONG,
    b_releaseDate   VARCHAR(20),
    b_condition     VARCHAR(20),
    PRIMARY KEY (b_bookId)
);
```

- Spring 애플리케이션 실행 시 자동 호출되는 SQL 스크립트

---

### root-context.xml 설정

### (1) Namespace 설정

```xml
<beans xmlns="http://www.springframework.org/schema/beans"
       xmlns:context="http://www.springframework.org/schema/context"
       xmlns:jdbc="http://www.springframework.org/schema/jdbc"
       xsi:schemaLocation="...">
```

- `jdbc`, `context` Namespace를 사용하기 위해 root 요소에 선언

### (2) db.properties 로드

```xml
<context:property-placeholder location="classpath:db.properties"/>
```

- 배포 환경별로 변경될 수 있는 설정을 외부 파일로 분리

### (3) DataSource 생성

```xml
<bean id="dataSource" class="org.springframework.jdbc.datasource.DriverManagerDataSource">
    <property name="driverClassName" value="${datasource.driverClassName}" />
    <property name="url"             value="${datasource.url}" />
    <property name="username"        value="${datasource.username}" />
    <property name="password"        value="${datasource.password}" />
</bean>
```

- `${...}` 플레이스홀더로 db.properties 값 참조
- `DriverManagerDataSource`는 `javax.sql.DataSource` 구현체

### (4) 데이터베이스 초기화

```xml
<jdbc:initialize-database data-source="dataSource">
    <jdbc:script location="classpath:db/create-db.sql" />
    <jdbc:script location="classpath:db/insert-data.sql" />
</jdbc:initialize-database>
```

- Spring 애플리케이션 시작 시 테이블 생성 및 초기 데이터 삽입

---

### servlet-context.xml - JdbcTemplate 설정

```xml
<beans:bean id="jdbcTemplate" class="org.springframework.jdbc.core.JdbcTemplate">
    <beans:property name="dataSource" ref="dataSource" />
</beans:bean>
```

- Repository에서 JDBC를 통해 SQL을 실행하는 핵심 클래스

---

### H2Console 서블릿 설정 (`web.xml`)

```xml
<servlet>
    <servlet-name>H2Console</servlet-name>
    <servlet-class>org.h2.server.web.JakartaWebServlet</servlet-class>
    <load-on-startup>1</load-on-startup>
</servlet>
<servlet-mapping>
    <servlet-name>H2Console</servlet-name>
    <url-pattern>/console/*</url-pattern>
</servlet-mapping>
```
